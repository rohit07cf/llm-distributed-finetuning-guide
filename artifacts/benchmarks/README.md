# Benchmark Results

This directory stores training benchmark outputs generated by `src/benchmark.py`.

## Files

- `sample_benchmark_results.json` — Example output schema showing the expected format
- `benchmark_*.json` — Individual benchmark run results
- `comparison_report.json` — Side-by-side comparison across configurations

## Generating Benchmarks

```bash
bash scripts/benchmark_training.sh
```

## Output Schema

Each benchmark result contains:

| Field | Description |
|---|---|
| `config_name` | Name of the training config |
| `method` | Training method (lora, qlora, deepspeed_z3) |
| `num_gpus` | Number of GPUs used |
| `effective_batch_size` | batch_size × grad_accum × num_gpus |
| `total_training_time_sec` | Wall-clock training time |
| `avg_step_time_sec` | Average time per training step |
| `tokens_per_sec` | Training throughput in tokens/second |
| `samples_per_sec` | Training throughput in samples/second |
| `gpu_memory_peak_mb` | Peak GPU memory usage |
| `gpu_utilization_avg_pct` | Average GPU utilization |
| `training_loss_final` | Final training loss |
